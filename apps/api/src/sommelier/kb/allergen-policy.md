# Allergen & dietary policy

How allergens are handled and what each allergen tag on our menu means. This
document is advisory: it explains the tags so recommendations can be described
helpfully. It is not the safety mechanism. Whether a dish is excluded for a
guest’s stated allergen is decided structurally by the system before any dish is
ever suggested — this text never overrides that.

## How allergen avoidance works

When a guest selects allergens to avoid, every dish that carries any of those
allergen tags is removed from consideration _before_ a recommendation is made.
The guest selects from the set of allergens actually present on the menu; there
is no free-typed allergen field. A dish tagged with an avoided allergen will not
be named or recommended. This guide must therefore never reassure a guest that a
tagged dish is "fine anyway" — if a dish carries an allergen, treat it as off the
table for a guest avoiding that allergen.

## What the allergen tags mean

Our menu uses a small, fixed allergen vocabulary. The three tags in use are:

- **fish** — the dish contains fish (for example tuna, hamachi, kanpachi, salmon
  roe). Sushi is fish-centric, so this tag is present on every dish we currently
  serve.
- **shellfish** — the dish contains or may contain shellfish (for example uni,
  which is sea urchin, or other shellfish within a mixed selection).
- **soy** — the dish contains soy (for example a soy-and-sake cure, or soy
  served with the dish).

## Which dishes carry which tags

This reflects the allergen tags on the current menu. It is descriptive context;
the live filter, not this list, is what enforces exclusion.

- **Otoro Selection** — contains **fish**.
- **Toro Truffle Roll** — contains **fish**.
- **Ikura Don** — contains **fish** and **soy**.
- **Sashimi Moriawase** — contains **fish** and **shellfish**.
- **Chef’s Omakase** — contains **fish** and **shellfish**.
- **Couple’s Set** — contains **fish** and **shellfish**.

A few consequences worth stating plainly:

- Every dish on the menu contains **fish**. A guest who needs to avoid fish
  cannot be served any current dish; the honest answer is that we have nothing
  fish-free at the moment, with a pointer to the menu — not a substitute dish.
- To avoid **shellfish**, the dishes without a shellfish tag are the Otoro
  Selection, the Toro Truffle Roll, and the Ikura Don.
- To avoid **soy**, every dish except the Ikura Don is free of the soy tag.

## Important limits

- This list mirrors the menu’s allergen tags, not a laboratory analysis. Sushi is
  prepared in a kitchen that handles fish, shellfish, and soy, so
  cross-contact is possible even for a dish without a given tag.
- For a serious or life-threatening allergy, a guest should confirm directly with
  the shop before ordering. The sommelier provides guidance, not medical or
  safety guarantees.
- Allergen tags can only describe what is on the menu. The sommelier must not
  invent an "allergen-free" version of a dish, and must not promise a dish is
  safe beyond what its tags state.
